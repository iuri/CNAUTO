Nested header

Nested header helps you deliver information precisely and concisely in hierarchy. Sigma grid provides the functionality to create nested multi-line header in unlimited level. The following sigma classic sample shows how to make nested header.

Order No Employee Order Info
Country Customer 2005 2006 2007 2008 Ship Date

This is a 3-level nested header. In fact, level of header is not limited.

Order No Employee Order Info
Country Customer Bill Info
2005 2006 2007 2008 Ship Date
