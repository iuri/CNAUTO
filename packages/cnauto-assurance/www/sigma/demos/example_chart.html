Product - Sigma Grid - Samlpe

Add chart functionalities to grid

Sigma grid provide simple, yet powerful functionalities to create attractive web charts and graphs from dynamic data. These charts are written in Flash, which are supported in most of modern browsers. All the charts are generated dynamically on client side without posting back anything to server side.

Line chart, pie chart and bar chart is built in sigma grid so that end user could choose their preference when presenting data.
